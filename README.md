# NVLink & NVSwitch — How NVIDIA Builds One Big GPU Out of Many

A deep dive into NVIDIA's scale-up interconnect from NVLink 1 (Pascal, 2016) to NVLink 5 (Blackwell NVL72, 2024). Covers NVSwitch fabrics, NVLink-C2C coherent CPU-GPU links (Grace-Hopper, Grace-Blackwell), the rack-scale NVL72 superpod, and how all of this compares to PCIe P2P and InfiniBand. Includes an interactive fabric planner.

**Live site:** https://brendanjameslynskey.github.io/NVIDIA_GPU_05_NVLink_NVSwitch/

Part of the [NVIDIA GPU Architectures series](https://github.com/BrendanJamesLynskey/LLMs#nvidia-gpu-architectures).
